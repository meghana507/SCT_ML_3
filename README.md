# Image Classification: Cats vs. Dogs

## Project Overview
A computer vision project that implements a **Support Vector Machine (SVM)** to classify images as either a cat or a dog. This demonstrates the application of traditional machine learning algorithms to image data.

## Dataset
* Source: Kaggle (Cats and Dogs Dataset)
* Data: Thousands of labeled images of pets.

## Methodology
1. **Image Processing:** Resizing images and converting them to grayscale or flattened arrays.
2. **Feature Extraction:** Extracting pixel intensities or using HOG (Histogram of Oriented Gradients).
3. **Classification:** Training an SVM model with a linear or RBF kernel.
4. **Testing:** Evaluating accuracy on a dedicated test set.

## Tech Stack
* Python
* OpenCV (for image processing)
* Scikit-learn (for the SVM algorithm)
